# Big-Lottery
Big-Lottery is a web application built with Node.js and Express. It uses EJS templating and has a MySQL database. The project exists to provide a lottery system with various features.

## How it works
The application uses Express.js to handle HTTP requests and responses. It connects to a MySQL database using the mysql2 package. The data flow involves user interactions with the web interface, which are handled by the server.js file and routed through the routes directory. The server uses various middleware packages like body-parser and cookie-parser to process requests.

## Tech Stack
* Node.js
* Express.js
* EJS
* MySQL
* Babel
* Nodemon
* Axios
* Socket.io

## Getting Started
To get started with the project, run the following commands:
npm install
npm run database
npm run start
These commands will install the dependencies, create the database, and start the server.
